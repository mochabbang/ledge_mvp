## 파서 — 타입 분류

| 입력 | 기대 type | 이유 |
|---|---|---|
| `광고비 12만` | expense | 지출 키워드: 광고 |
| `점심 1.2만` | expense | 지출 키워드: 점심 |
| `외주 50만원` | income | 수입 키워드: 외주 |
| `강의료 30만` | income | 수입 키워드: 강의 |
| `급여 200만` | income | 수입 키워드: 급여 |
| `교재비 35000` | expense | 기본값 expense |
| `임대료 100만` | expense | 지출 키워드: 임대 |
| `매출 80만` | income | 수입 키워드: 매출 |
| `입금 50만` | income | 수입 키워드: 입금 |
| `커피 5천원` | expense | 지출 키워드: 커피 |
| `abc 50만원` (키워드 없음) | expense | 기본값 expense |
| `외주비 점심 50만` (충돌) | expense | 충돌 → 지출 우선 |

---

## typeOverride — API 레벨

| text | typeOverride | 저장 type |
|---|---|---|
| `외주 50만원` | (없음) | income (파서 결과) |
| `외주 50만원` | `"expense"` | expense (override 우선) |
| `점심 1.2만` | `"income"` | income (override 우선) |
| `abc 50만` | (없음) | expense (기본값) |

## UI 동작

| 시나리오 | 기대 결과 |
|---|---|
| 텍스트 입력 시 파싱 미리보기 노출 | 수입/지출 뱃지 + 금액 표시 |
| 미리보기 수입/지출 버튼 탭 | 색상 전환 (초록↔빨강) |
| 텍스트 변경 시 override 초기화 | 파서 결과로 재판단 |
| `needs_confirm=true` | "금액 확인" 뱃지 표시 |
| 금액 미감지 텍스트 입력 후 Enter | 미리보기 없음, 저장 시 "금액을 찾지 못했어요" |
| 과거 날짜 선택 후 저장 | 해당 월의 내역에 포함 |
| 미래 날짜 선택 시도 | 날짜 선택기 max 제한으로 불가 |
| 월 네비게이터 `›` (현재 월) | 비활성 (`disabled`) |
| 내역 항목 클릭 | 인라인 수정 패널 열림 |
| 삭제 첫 클릭 | "확인" 버튼으로 변경 |
| 삭제 확인 클릭 | 삭제 후 목록 갱신, "삭제됨" 토스트 |
| 등록자 닉네임 저장 후 거래 입력 | 내역에 닉네임 뱃지 표시 |
