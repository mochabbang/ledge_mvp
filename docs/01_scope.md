# 01. 범위/목표/비기능

## 목표
- 사용자가 **월 단위 순수익 목표**를 설정한다.
- 웹 대화 입력(1줄) → **자동 저장**
- 입력 즉시 **실시간 파싱 미리보기** (수입/지출 + 금액 확인 후 Enter 제출)
- 대시보드에서 **실시간 집계**를 즉시 확인:
  - 수입 / 지출 / 순수익 / 달성률 / 남은 금액
- 잘못 분류된 경우 **입력창 내 토글 버튼**으로 즉시 수정 후 저장
- **가족 공유**: 누가 등록했는지 닉네임으로 표시

## MVP 범위(In)
- 월 목표(Goal) 설정/수정 (온보딩 페이지)
- 사용자 닉네임 설정 (온보딩, `user_metadata.display_name`)
- 자유 텍스트 입력 → 실시간 파싱 → 수입/지출 미리보기 → Enter 저장
- 날짜 선택 입력 (과거 날짜 등록 지원)
- 월 네비게이터 (이전/다음 월 조회, 미래 월 잠금)
- KPI 카드: 목표 / 순수익 / 달성률 / 총수입 / 총지출
- 진행바 (달성률 시각화)
- 일별 수입·지출 바 차트 (호버 툴팁 포함)
- 최근 내역 리스트 (날짜별 그룹, 일별 소계, 최대 30건)
- 인라인 수정 (아코디언 토글 — 내용/타입/금액/저장/삭제)
- 삭제 2단계 확인 (삭제 → 확인 클릭)
- 등록자 뱃지 (`registered_by`) 표시
- 사용자 인증/권한: Supabase Auth + RLS

## MVP 제외(Out)
- Undo(5초) — 삭제 기능으로 대체
- 제안 카드(SuggestionCard) — 입력창 실시간 미리보기로 대체
- 카테고리 관리 화면 (추가/편집)
- 태그/메모/첨부/영수증 OCR
- 통계 심화 (카테고리 파이, 예산 등)
- 자동 학습/추천 (ML/LLM) — **파싱 LLM 영구 금지**
- 다중 통화/환율

## 비기능 요구사항
- 모바일에서 입력 제출 후 **1초 내 체감 반응**
- 저장 실패 시 즉시 토스트 알림
- 모든 데이터는 **user_id 기준 완전 분리(RLS)**
- 파싱: 정규식/키워드만 (LLM 금지, 비용 0원)
